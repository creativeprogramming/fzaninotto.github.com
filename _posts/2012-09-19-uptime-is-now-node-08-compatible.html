--- layout: post title: Uptime Is Now Node 0.8 Compatible published: true ---

I've just released Uptime v1.4 and v2.0rc0. They share the exact same features, but one is for Node 0.6.x, and the second is for Node 0.8.x. Uptime 1.4 is the last release for the old Node, and Uptime 2.0rc0 will become stable as soon as Express goes stable. Both are backwards compatible with previous versions.

All versions are available on npm, or as tags on GitHub.

Feel free to open issues on the GitHub tracker if you find any regression in v2.0 - just don't forget to mention you're on the Node 0.8 version.
